Wheel Exporter

Downloads all wheel artwork extracted directly from the SWF files — no browser rendering required. Covers all 5 views (FF, BF, FR, BR, R) × 154 designs = 770 SWFs.

SVG ZIP — recommended

Each wheel bitmap converted to SVG with the original JPEG embedded and alpha mask applied. Ready to open in Inkscape, Illustrator, or a browser.

Multi-layer wheels (designs 50+) produce one SVG per bitmap layer.
Structure: wheels/wheelFF/wheelFF_1.svg, wheels/wheelBF/… etc.

Raw Bitmaps ZIP

Original JPEG files + separate greyscale alpha PNG masks, exactly as stored in the SWF. Useful if you want to do your own compositing.

JPEG3 bitmaps → .jpg + _alpha.png
Lossless2 bitmaps → .png (RGBA, already merged)
